#BOOK MANAGEMENT IN THE LIBRARY

# 📌 Description
- This project is a simple library book management system written in Python.
- It allows users to manage books in a library such as adding, deleting, updating, searching books.

# Main function
- Add new book
- Edit book information
- Search for books
- Delete a book
- Borrow a book
- Return book
- File handling

# Technologies
- Python
- Python OOP
- JSON
- Git and github

# Install and run programs
1. clone repository
###bash:
git clone https://github.com/HLiem-200570/Book-management-in-the-library
- cd Book-management-in-the-library
- python Main.py
